Install npm dependencies

    npm install

## Run project

This project uses [Gulp](http://gulpjs.com/) as build system. The default task is "dev".

    gulp [dev]

## Log files

Logging is implemented with [log4js](https://github.com/nomiddlename/log4js-node) and files are stored
in `./logs`
